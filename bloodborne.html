BLOODBORNE - Acceptable BL4 Great One Criteria

  • BL4 NG+6
  • BL4 NG+ & No Chalice Dungeon Gems or Runes
  • BL4 No Rolling/Quickstepping/Blocking/Beast Blood Pellets run - Parrying is allowed, especially versus Gehrman
  • BL4 Perma-Death run - Record all of the run
  • BL4 Not Hit run & Any% - Record all of the run
  • BL4 Blindfolded run - Not allowed to see the screen, needs a clear webcam stream to prove
  • BL4 +0 Weapons
  • BL4 All Bosses Speedrun under 1 hours 30 minutes - Mandatory: Includes The Old Hunters
  • BL4 Fist only
  • BL4 Torch only - Either Torch counts

BLOODBORNE - BL4 Great One Banned Equipment

  • Usage of Chalice Dungeon gems or Runes in base NG runs - They're too strong.
